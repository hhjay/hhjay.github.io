undefined-null的区别

huiujie's blog
2018-03-28
js

undefined

  • Undefined是一个数据结构
  • 继承至Null
  • 表示一个“无”的原始值（初始值）
  • 产生undefined值的

null

  • 表示一个没有任何引用的空对象（空对象指针） .

NaN

  • Not a Number
  • NaN与所有的值都不相等，包括它自己
  • 可用isNaN函数来判断

  • 类型不同

  • undefined不是有效的JSON值，null是

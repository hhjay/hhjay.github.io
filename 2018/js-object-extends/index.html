对象

  • 一切事物皆对象
  • 对象具有封装和继承特性

原型链继承

  • 对象（object）依赖类（class）来产生，而基于原型链的面向对象方向中，对象依赖构造器（constructor）利用原型链（prototype）构造出来的
  • 被继承的属性和方法都放在prototype中
  • 子继承者自己的属性和方法也放在自己的prototype
  • 子继承者的prototype对象的proto属性指向被继承的prototype
